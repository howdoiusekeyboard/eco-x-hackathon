üåæ Farmer Waste Submission Journey

1
Navigate to "Log Waste"
Farmer opens the PWA app and clicks on the prominent "Log Waste" button from their personalized dashboard.
2
Capture Waste Photo
Camera interface opens automatically. Farmer takes a clear photo of agricultural waste (max 5MB) which gets uploaded to Firebase Storage.
3
Auto-Capture GPS Location
App automatically captures precise GPS coordinates using device location API for accurate distance calculations.
4
Fill Waste Details Form
Select waste type (Rice Straw/Wheat Residue), enter quantity in kg, moisture level %, and current season (Rabi/Kharif/Zaid).
5
AI Matching Triggered
Cloud Function automatically triggers, AI analyzes data, finds optimal industry match in 2-5 seconds, and displays success screen with match details and environmental impact.

üß† AI Decision-Making Process

üéØ Waste Type Compatibility
40%
Analyzes exact match between farmer's waste type and industry demand. Rice straw for biogas plants, wheat residue for animal feed, etc.
üìç Proximity Logistics
30%
Calculates transportation efficiency using Haversine formula. Prioritizes matches within 15km for optimal logistics and reduced carbon footprint.
üìä Demand Capacity Fit
20%
Evaluates if industry can handle the waste quantity. Checks current capacity, processing capabilities, and demand urgency.
üí∞ Price Optimization
10%
Ensures fair pricing for farmers while maintaining industry profitability. Factors in market rates and transportation costs.

Gemini 2.0 Flash Thinking Process

The AI agent processes all factors simultaneously, generating a match score (0-100) for each potential industry. The highest scoring match is selected with explainable reasoning provided to both farmer and industry. Average decision time: 2-5 seconds
